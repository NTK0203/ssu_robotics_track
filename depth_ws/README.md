# ROS2 Depth Camera Practice (depth_ws)

이 패키지는 Intel RealSense Depth 카메라와 OpenCV를 활용하여 가상 및 실제 환경의 물체 거리를 측정하고, 3D 좌표 변환 및 캘리브레이션을 통해 카메라로 인식한 물체를 로봇이 직접 파지(Pick & Place)하는 지능형 시각 제어 시스템을 구축한 실습 기록입니다.

---

## 1. 수업 내용 요약

### Depth 카메라의 원리 및 필요성
- **한계 극복**: 일반적인 2D 카메라는 색상(RGB) 정보만 제공하므로 로봇이 물체까지의 실제 거리를 파악할 수 없습니다.
- **공간감 부여**: Depth 카메라는 적외선 패턴 투사나 스테레오 비전 방식을 통해 각 픽셀마다 거리 데이터($Z$)를 포함한 RGB-D 정보를 제공합니다. 이를 통해 로봇에게 주변 환경에 대한 정밀한 '공간감'을 제공할 수 있습니다.

### 3D 좌표 변환 (Pixel to 3D)
- **수학적 모델**: 이미지상의 2D 픽셀 좌표($u, v$)와 깊이 값($depth$)을 결합하여 카메라 좌표계 기준의 실제 3D 위치($X, Y, Z$)를 계산합니다.
- **내부 파라미터(Intrinsic)**: 정확한 계산을 위해 카메라의 초점 거리($f_x, f_y$)와 주점($c_x, c_y$) 정보가 담긴 고유 값을 활용합니다.

### 카메라 캘리브레이션 (Calibration)
- **내부 파라미터 보정**: 렌즈 제작 공정에서 발생하는 미세한 왜곡을 수학적으로 보정하여 이미지의 정밀도를 높이는 과정입니다.
- **외부 파라미터(Extrinsic) 설정**: 로봇 베이스 좌표계와 카메라 좌표계 사이의 상대적 위치($T$)를 정의합니다. 이 변환 행렬($T_{base}^{camera}$)이 정확하게 설정되어야만 '카메라'로 본 물체 좌표를 '그리퍼'가 찾아갈 수 있는 좌표로 변환할 수 있습니다.

---

## 2. 실습 내용

### 주요 활용 패키지
- **realsense2_camera**: RealSense 하드웨어를 구동하고 정렬된 이미지/깊이 데이터를 발행하는 패키지입니다.
- **dsr_practice**: 수집된 시각 데이터를 처리하고 로봇 팔의 동작을 제어하는 로직이 통합된 패키지입니다.

### 주요 파일 및 구현 세부 사항
1. **`depth_click_viewer.py` (3D 좌표 측정 실습)**
   - **기능**: GUI 화면에서 마우스로 특정 물체를 클릭하면 해당 지점의 카메라 기준 실제 거리($X, Y, Z$, m 단위)를 실시간으로 계산하여 출력합니다.
   - **구현**: `sensor_msgs/Image` 토픽으로 수신되는 `uint16` 깊이 데이터를 m 단위로 환산한 뒤, 카메라 내부 파라미터를 적용하는 삼각측량 수식을 코드로 작성했습니다.

2. **`click_pick_node.py` (시각 기반 Pick & Place 구현)**
   - **기능**: 클릭한 픽셀의 3D 좌표를 로봇 베이스 좌표계로 변환하고, 해당 위치로 로봇 팔을 이동시켜 물체를 집어 올리는 동작을 수행합니다.
   - **구현**: 외부 파라미터 변환 행렬을 활용하여 $T_{base}^{target} = T_{base}^{camera} \cdot T_{camera}^{target}$ 연산을 수행하도록 설계했습니다. 계산된 최종 목표 포즈는 MoveItPy 인터페이스를 통해 로봇에 전달됩니다.

---

## 3. 실행 절차 및 검증

작성된 노드들을 통합하여 시각 기반 로봇 제어 시스템을 구동하고 성능을 검증합니다.

```bash
# 1. 두산 로봇 실행 (실물 로봇 연동 및 MoveIt 인터페이스 활성화)
ros2 launch dsr_bringup2 dsr_bringup2_moveit.launch.py mode:=real model:=m0609 host:=192.168.1.100

# 2. RealSense 카메라 구동 (Depth와 RGB 이미지의 정렬 기능 활성화)
ros2 launch realsense2_camera rs_align_depth_launch.py initial_reset:=true

# 3. Depth 기반 Pick & Place 노드 실행
ros2 launch dsr_practice click_pick_node.launch.py

# 검증 포인트
# - 화면 클릭 시 출력되는 Z값이 실제 측정 거리와 일치하는지 확인합니다.
# - 베이스 좌표계로 변환된 좌표가 RViz의 Planning Scene 상의 위치와 일치하는지 대조합니다.
# - 로봇이 물체를 잡으러 갈 때 그리퍼가 물체의 중앙 상단에 정확히 위치하는지 확인하여 캘리브레이션 오차를 점검합니다.
